# Homebrew tap for soma

[soma](https://github.com/radotsvetkov/soma) is a small, zero-dependency runtime
that governs AI agents and proves what they did.

```sh
brew install radotsvetkov/soma/soma
```

The formula installs the prebuilt binary attached to each soma release. See the
[soma repository](https://github.com/radotsvetkov/soma) for documentation.
